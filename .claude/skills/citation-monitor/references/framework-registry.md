## Scan additions protocol

When adding a new framework:

1. Publish the framework's canonical essay
2. Add a new entry above with canonical name, date, URL, anchors, variants
3. Note any misattribution risks (common phrases that would produce false positives)
4. Run the next scan and verify baseline is sensible — if scan returns dozens of pre-existing hits, the framework name is too generic and needs refinement

Never retroactively edit a framework's canonical name after publication — it breaks the citation log's continuity.
